【CSP】训练计划

2023/2/28 算法题 csp AOE 数组

该题表达的意思就AOE网络中任务的最早开始时间和最迟开始时间。而由于题目中明确标明了每项科目最多只依赖一项别的科目，且满足依赖科目的编号小于自己，因此在具体的实现过程中简单很多。

可以将整个任务网络看成一个树，子结点的最早开始时间依赖于父结点的最早开始时间，因此有：

early[i] = early[fa[i]] + cost[fa[i]]

对于最晚开始时间，可以参考AOE网中的计算方法，而在该题中则简化了一些。使用一个vector存储每个结点的子结点，那么该结点的最迟开始时间依赖于子结点中最早的最迟开始时间（因为不能影响后续任务的进行）：

late[i] = min(late[son[i]]) - cost[i]

在这里需要注意的是，最后一个完成的任务（也就是树的叶子结点）在计算时，需要+1，即：

late[i] = n - cost[i] + 1

AC代码：

#include <iostream>
#include <cmath>
#include <vector>
using namespace std;


int n, m;
int depend[1005], cost[1005];
int early[1005], late[1005];
vector<int> son[1005];
int keyPathLen = 0;

int main()
{
    cin >> n >> m;
    for (int i = 1; i <= m; i++)
        cin >> depend[i];
    for (int i = 1; i <= m; i++)
        cin >> cost[i];  

    for (int i = 1; i <= m; i++)
        son[depend[i]].push_back(i);

    int canFinish = 1;

    // 最迟开始时间直接就是父节点的开始时间 + 父节点的消费时间
    for (int i = 1; i <= m; i++)
    {
        // 根据题意，父节点只有一个
        if (depend[i] == 0)
            early[i] = 1;
        else
            early[i] = early[depend[i]] + cost[depend[i]];
        if (early[i] + cost[i] - 1 > n)
            canFinish = 0;
    }

    //cout << "keyPathLen: " << keyPathLen << endl;
    for (int i = 1; i <= m; i++)
        cout << early[i] << " ";
    cout << endl;
    // 求最迟时间
    if (canFinish)
    {
        for (int i = m; i >= 1; i--)
        {
            // 这里需要+1，具体可以找几个例子
            if (son[i].size() == 0)
                late[i] = n - cost[i] + 1;
            else
            {
                // 最晚时间是不影响子节点的开始时间，需要获得子节点中最小的最迟开始时间
                int minLen = INT_MAX;
                for (int j = 0; j < son[i].size(); j++)
                    minLen = min(minLen, late[son[i][j]]);
                late[i] = minLen - cost[i];
            }
        }
        for (int i = 1; i <= m; i++)
            cout << late[i] << " ";
        cout << endl;
    }
    system("pause");
}
