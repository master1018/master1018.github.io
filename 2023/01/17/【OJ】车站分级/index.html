【OJ】车站分级

2023/1/17 算法题 拓扑排序

原题链接

P1983 [NOIP2013 普及组] 车站分级 - 洛谷 | 计算机科学教育新生态 (luogu.com.cn)

输入输出样例

输入 #1复制

9 2 
4 1 3 5 6 
3 3 5 6

输出 #1复制

2

输入 #2复制

9 3 
4 1 3 5 6 
3 3 5 6 
3 1 5 9

输出 #2复制

3

题目中关键点：

如果这趟车次停靠了火车站x，则始发站、终点站之间所有级别大于等于火车站x的都必须停靠

以样例1来说，第一趟火车停在了1，3，5，6，则站点2、4的级别均小于1、3、5、6；第二趟车停在了3、5、6，则4的级别小于3、5、6。那么我们可以得到这样的级别关系：

{2, 4} < {1, 3, 5, 6}
{4} < {3, 5, 6}

只知道2、4分别和1、3、5、6的关系，但不知道2和4的具体关系，以及1、3、5、6的关系，这种类型的排序正是拓扑排序，因此这道题显而易见的要使用到拓扑排序。

但是知道拓扑排序很简单，但重要的是怎么从拓扑排序中获得这些车站的级别关系。根据题目要求，我们最终要获得拓扑排序的最长链才行，以此来设计的车站级别个数才会最少。

在这里采用一个动态规划的思路，以下图为例：

设f(i)为从入度为0的结点到i结点的最长距离（经过的结点个数最多），那么在拓扑排序的中途，我们用f数组维护这样一个长度最大值。比如上图中，方位结点1后，去放分结点2和4（即2、4入队列），然后更新f(2)和f(4)为1。接着访问结点2指向的结点，会访问到结点4，此时有f(4) = max(f(4), f(2) + 1)，最终结果是更新f(4)为2，相关的代码部分如下：

for (int j = 0; j < g[x].size(); j++)
{
    // 删去x -> t这条边
    int t = g[x][j]; 
    f[t] = max(f[t], f[x] + 1);  // 更新到达结点t所经过的最长的路径
    du[t]--;
    if (!du[t]) q.push(t);
}

找到最长的路径后，路径的长度 + 1即为最终的答案（+1细品）。

完整代码如下：

#include <iostream>
#include <cstring>
#include <cstdlib>
#include <vector>
#include <queue>
using namespace std;

int f[1005] = {0};

vector<int> g[1005];  // 邻接表
int du[1005], n, m, L[1005];
int visti[1005][1005] = {{0}}; // visit用于判重，否则会MLE

bool toposort()
{
    memset(du, 0, sizeof(du));
    for (int i = 0; i < n; i++)
        for (int j = 0; j < g[i].size(); j++)
            du[g[i][j]]++;
    int tot = 0;
    queue<int> q;
    for (int i = 1; i <= n; i++)
        if (!du[i]) q.push(i), f[i] = 0;
    while (!q.empty())
    {
        int x = q.front(); q.pop();
        L[tot++] = x; 
        for (int j = 0; j < g[x].size(); j++)
        {
            // 删去x -> t这条边
            int t = g[x][j]; 
            f[t] = max(f[t], f[x] + 1);
            du[t]--;
            if (!du[t]) q.push(t);
        }
    }
    if (tot == n) return 1;
    return 0;
}

int main()
{
    cin >> n >> m;
    for (int k = 1; k <= m; k++)
    {
        int s;
        cin >> s;
        int a[1005] = {0};
        int pos[1005] = {0};
        for (int i = 1; i <= s; i++)
            cin >> a[i], pos[a[i]] = 1;
        for (int i = a[1]; i <= a[s]; i++)
            if (!pos[i]) 
                for (int j = a[1]; j <= a[s]; j++)
                    if (pos[j] && !visti[i][j]) g[i].push_back(j), visti[i][j] = 1;

    }

    toposort();
    int ans = 0;
    for (int i = 1; i <= n; i++)
        ans = max(ans, f[i]);
    cout << ans + 1 << endl;

    //system("pause");
}
