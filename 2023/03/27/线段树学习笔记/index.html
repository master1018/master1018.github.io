线段树学习笔记

2023/3/27 算法 线段树

线段树模版题目：

https://www.luogu.com.cn/problem/P3372

线段树可以很有效的解决一些区间查询的问题（csp第4题，蓝桥杯的一些题目非常常见），如果不会线段树，遇到一些算法题目会被薄纱qwq。

线段树的主要思想就是二分，可以将一些O(N)的算法降低为O(logN)级别的，面对一些数据量为1e6的题目，可以很有效的解决。

线段树的写法有结构体和数组的形式，目前只打算学习一下数组的写法（以便于能够快速的在赛场上混个几分）。

建树

初始化直接全局变量就行了，不用怎么进行操作。

void build(int l, int r, int pos, int val)
{
    if (l == r)
    {
        // 建树的时候，叶子节点的取值，可以根据题目进行不同的处理
        //tree[pos] = val;
        cin >> tree[pos];
        return;
    }
    int mid = (l + r) / 2;
    build(l, mid, 2 * pos, val); // 左孩子
    build(mid + 1, r, 2 * pos + 1, val); // 右孩子
      // 记录区间的和，根据题目进行修改
    tree[pos] = tree[2 * pos] + tree[2 * pos + 1]; 
}

如果题目中输入的数组比较复杂，可以将一维数组改为结构体数组，大致写法相似。

单点更新

void update(int l, int r, int idx, int pos, int val)
{
    if (l == r)
    {
        // 更新叶子结点的方式
        // 可以根据题目进行其他更改
        tree[pos] += val;
        return;
    }
    int mid = (l + r) / 2;
    // 左孩子递归
    if (idx <= mid) update(l, mid, idx, 2 * pos, val);
    // 右孩子递归
    else update(mid + 1, r, idx, 2 * pos + 1, val);

    tree[pos] = tree[2 * pos] + tree[2 * pos + 1];
}

该部分在idx和mid判断的时候，根据idx的大小来决定从左子树找值还是从右子树找值，找到之后进行更新。（单点操作时在数据量大的时候会超时，但对于一些题目的混分，可以较为轻松的写出）

范围更新

单点更新在数据量较大的时候，每次的更新都会花费大量的时间，对此可以采用懒惰标记的方式。在每次更新时，在大区间对应的位置添加标记，表示这个区间的数据变化，然后在每次查询的时候，由于查询时递归的，便可以将标记一层层的带下去，从而更新所有结点的值。

该部分的思想有点类似于前缀和更新，更新[l, r]的数组元素值+1，通常只需在前缀和的l打上+1标记，在r + 1打上-1标记，然后一次遍历更新前缀和即可。

void push_down(ll l, ll r, ll pos)
{
    if (lazy[pos])
    {
          // 将懒惰标记带给子树
        lazy[2 * pos] += lazy[pos];
        lazy[2 * pos + 1] += lazy[pos];

        ll mid = (l + r) / 2;
          // 更新子树的值
        tree[2 * pos] += (mid - l + 1) * lazy[pos];
        tree[2 * pos + 1] += (r - mid) * lazy[pos];
        lazy[pos] = 0; 
    }
}

void update_range(ll l, ll r, ll x, ll y, ll pos, ll val)
{
    if (x <= l && r <= y)
    {
          // 更新懒惰标记以及区间和
        lazy[pos] += val;
        tree[pos] += (r - l + 1) * val;
        return ;
    }

    push_down(l, r, pos);
    ll mid = (l + r) / 2;
    if (mid >= x) update_range(l, mid, x, y, 2 * pos, val);
    if (mid < y) update_range(mid + 1, r, x, y, 2 * pos + 1, val);
    tree[pos] = tree[2 * pos] + tree[2 * pos + 1];
}

查询

ll query_range(ll l, ll r, ll x, ll y, ll pos)
{
    if (x <= l && r <= y) return tree[pos];
      // 每次查询都对懒惰标记进行处理
    push_down(l, r, pos);
    ll mid = (l + r) / 2;
    ll sum = 0;
    if (mid >= x) sum += query_range(l, mid, x, y, 2 * pos);
    if (mid < y) sum += query_range(mid + 1, r, x, y, 2 * pos + 1);
    return sum;
}
